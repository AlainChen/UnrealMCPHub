## Maintenance Rule

如果后续再增加文档，建议优先判断它属于哪一类：

- `Current`
  当前仍然会指导实际工作
- `Snapshot`
  某一阶段的状态留档
- `Archive`
  已经被吸收的过程稿或中间稿

不要再让 `docs/` 根目录堆积新的中间过程文档。默认都应先判断是否应该进入 `unreal-ai-playbook/` 的 `Current / Snapshot / Archive` 结构。
