# Review Checklist

Check these before considering a task complete:

- Did the task stay inside approved scope?
- Did it avoid forbidden paths?
- Did validation actually run?
- Is the change summary complete?
- Were shared assets or maps touched?
- Is the next step clearly smaller than the remaining problem?

For benchmark-lite runs also check:
- Was the scenario level appropriate?
- Did the agent complete the task without manual rescue?
- Were logs and outcomes recorded clearly enough to compare later?
